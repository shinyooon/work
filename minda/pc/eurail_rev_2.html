유레일 셀렉트패스 4개국

유레일 셀렉트 패스는 국가에 따라서 3개의 요금 레벨이 적용됩니다.
  • 국가별 요금 레벨 Value H, Value M, Value L 적용
  • Value H (20점): 독일, 이탈리아, 프랑스, 스위스, 오스트리아
  • Value M (5점): 스페인, 베네룩스, 체코, 스웨덴, 덴마크, 노르웨이, 헝가리
  • Value M (5점): 스페인, 베네룩스, 체코, 스웨덴, 덴마크, 노르웨이, 헝가리
  • Value L (1점): 슬로바키아, 포르투갈, 핀란드, 루마니아, 불가리아, 그리스, 터키 등
  • 선택한 국가에 따라서 합산 점수가 산출되고 이에 따라 Price H, Price M, Price L 요금 적용

예) 46점 이상 Price H / 11~45점 Price M / 10점 이하 Price L 요금 적용

  • 프랑스, 독일, 이탈리아 스위스 선택 => 80점 => Price Level H
  • 오스트리아, 체코, 프랑스, 독일 => 65점 => Price Level H
  • 오스트리아, 베네룩스, 핀란드, 독일 선택 => 46점 => Price Level H
  • 불가리아, 그리스, 이탈리아, 터키 선택 => 23점 => Price Level M
  • 불가리아, 그리스, 루마니아, 터키 선택 => 4점 => Price Level L
